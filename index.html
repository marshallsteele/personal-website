Peacasa Snacks Website

While working at Peacasa Snacks Inc., I had the opportunity to update their Shopify website using the updated branding. After being sent the initial designs in Adobe XD, I organized them into pages and sections to make development easier. I designed the website iteratively with feedback from the designer one page at a time. Maintaining a responsive layout for the product carousel on the front page and in the mobile navigation menu was the toughest technical challenge for this project. I had fun learning liquid as I developed as this was my first time developing Shopify website.

This is the main website for Peacasa Snacks Inc. providing a landing page for any potential customers. The front page quickly shows the products they have available and some major outlets that sell them. The Find a Store page allows users to locate the nearest retailer using an interactive map while also providing them online retail options. The About and FAQ pages provide a little more detail about the company and their products with a fun a playful aesthetic.

GW2 Legendary Price Check

I first designed this webpage using Figma which helped the aesthetic remain consistent and clear. It also helped me organize my React components ahead of time reducing duplication of code and keeping data organized. This tool also integrates with an API to display up to date information.

GW2 Legendary Price Check is a tool for players to quickly reference current prices of legendary items. For a given item, the player can refresh the card with current data and quickly calculate percentages of the total sale price. This is particularly useful for player to player trading as these percentages are often used instead of the full price. The card layout allows users to search and compare prices of many different items at once.
